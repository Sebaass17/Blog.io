Esta es la primera hoja. A partir de aquí se archivan las entradas del cuaderno: proyectos, ideas sueltas, cosas que voy aprendiendo.

Cómo está montado

Cada entrada es un archivo HTML dentro de /posts, con el mismo cajetín arriba (título, fecha, categoría, número de hoja) que en los planos de dibujo técnico. Para añadir una hoja nueva: duplica este archivo, cambia los datos del cajetín y añade el enlace en index.html.

Siguiente paso

Subir esto a un repositorio en GitHub y activar GitHub Pages.
